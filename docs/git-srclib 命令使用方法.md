# git-srclib 命令使用方法
    git-srclib 是一个 Git 命令扩展，它的使用方法类似于其它的 Git 命令。


## git srclib pull
#### git srclib pull -all
#### git srclib pull [import_name]
    这个命令用于签出/更新目标文件夹中的源代码。

## git srclib status
#### git srclib status -all
#### git srclib status [import_name]
    这个命令用于显示目标源码的版本等相关信息。
